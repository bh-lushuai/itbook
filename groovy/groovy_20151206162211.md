#### 一、数组的定义及赋初值
在Groovy语言中，数组的定义和Java语言中一样。

	    def a = new String[4]
	    def nums = newint[10]
	    def objs = new Object[3]

然后赋值也一样：

	a[0] = 'a'
	a[1] = 'b'
	a[2] = 'c'
	a[3] = 'd'

所不同的在于在数组定义的时候赋初值。

	在Java语言里，对一个字符串数组这样定义：
	String[] strs = new String[]{'a','b','c','d'};

而在Groovy语言中，对一个字符串数组则需要这样定义：

	def strs = ['a','b','c','d'] as String[]


#### 二、数组的遍历
在Groovy语言中，对数组的遍历方法很多，常用的是使用each方法：

	a.each{

	println it
	}

当然，你也可以使用增强for循环：

	for(it in a)
	{
	println it
	}

你还可以使用如下的遍历方式：
	(0..<a.length).each{
	println a[it]
	}
	···
