# 应用切换工具Manico
发布日期 2016年3月16日

软件主页[http://manico.im](http://manico.im) 

MAC 自带的CMD + Tab切换应用的效果，并不好，需要多次CMD + Tab才能切换到想要的应用，而且能切换的应用也比较少。Manico 是一个非常容易使用的 App，当您需要切换至一个 App 时，按住 Option 键和对应的目标数字，您将被立马切换至对应的 App。如果这个 App 还没有启动，Manico会先启动它。
